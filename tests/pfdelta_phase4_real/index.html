1. Methodology

1.1 Real Load Profile Integration

Load profiles from Open Power System Data (OPSD) replace the synthetic ShockGenerator seasonal variation. For each simulation step, the real hourly load value (normalized to [0, 1]) is mapped to a scaling factor applied to all IEEE 118-bus loads:

factor = 0.92 + normalized_profile[step] √ó 0.16   # ‚Üí [0.92, 1.08] on base 1.75√ó

The ShockGenerator local zone perturbations (¬±3‚Äì12%) are retained on top, adding realistic local variability.

5. Claims and Limitations

What this study claims:
  • inZORi FrozenElite maintains higher S3 convergence than NR standard on real OPSD load profiles from UA, DE, and FR
  • The advantage is statistically significant (CI95 non-overlapping) across all three countries
  • The algorithm is robust across structurally diverse European grids
  • Under extreme N-4 conditions (Ukraine 2015 scenario), all methods degrade equally ‚Äî the advantage shrinks to ~0.2%
What this study does NOT claim:
  • Results on real TSO network topologies (CGMES) ‚Äî only IEEE 118-bus
  • Exact replay of historical blackouts ‚Äî only calibrated proxies
  • Comparison with commercial solvers (PSS/E, PowerFactory)
  • Operational certification or deployment readiness
